# Open Tetris

Rust で作られたクラシックなターミナル向けテトリスゲーム。[ratatui](https://github.com/ratatui/ratatui) と [crossterm](https://github.com/crossterm-rs/crossterm) を使用しています。

## 機能

- **SRS 回転システム** — 7 種類すべてのテトロミノに対応したウォールキックテーブル
- **7-bag ランダマイザー** — Fisher-Yates シャッフルで公平なピース配分を実現
- **ロック遅延** — 500ms のロック遅延、最大 15 回の移動リセット
- **ゴーストピース** — 落下先を表示する半透明プレビュー
- **スコア** — Single/Double/Triple/Tetris (100/300/500/800 × レベル)、ソフトドロップ 1pt、ハードドロップ 2pt/行
- **レベル進行** — 速度が 800ms から 50ms まで指数関数的に上昇
- **状態遷移** — メニュー → プレイ中 → 一時停止 → ゲームオーバー

## クイックスタート

```bash
cargo run
```

## 操作方法

| キー | アクション |
|------|------------|
| ← → | 左右移動 |
| ↑ | 時計回りに回転 |
| Z | 反時計回りに回転 |
| ↓ | ソフトドロップ |
| Space | ハードドロップ |
| P | 一時停止 / 再開 |
| Q | 終了 |
| Enter | 開始 / 決定 |

## 技術スタック

- **Rust** — システムプログラミング言語
- **ratatui** 0.29 — ターミナル UI レンダリング
- **crossterm** 0.28 — ターミナル入力・制御
- **rand** 0.8 — 乱数生成

## プロジェクト構成

```
src/
├── main.rs       — 端末初期化 + 60fps イベントループ
├── app.rs        — 状態遷移（メニュー/プレイ/一時停止/ゲームオーバー）
├── game.rs       — コアロジック：重力、ロック遅延、スコア
├── board.rs      — 10×20 グリッド、衝突判定、ライン消去
├── piece.rs      — 7 種テトロミノ、SRS 回転状態、ウォールキック
├── bag.rs        — 7-bag ランダマイザー
├── ui.rs         — ratatui 描画：ボード、サイドパネル、メニュー
├── input.rs      — crossterm キー → アクション マッピング
└── constants.rs  — グリッドサイズ、タイミング、スコア定数
```
