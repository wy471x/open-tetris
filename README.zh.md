# Open Tetris

基于 Rust 的经典俄罗斯方块终端游戏，使用 [ratatui](https://github.com/ratatui/ratatui) 和 [crossterm](https://github.com/crossterm-rs/crossterm) 构建。

## 特性

- **SRS 旋转系统** — 超级旋转系统，包含全部 7 种方块的标准墙踢偏移表
- **7-bag 随机算法** — Fisher-Yates 洗牌保证每 7 块每种方块恰好出现一次
- **锁定延迟** — 500ms 触底锁定延迟，最多允许 15 次移动重置
- **幽灵方块** — 半透明预览方块落点位置
- **计分系统** — 单消/双消/三消/四消 (100/300/500/800 × 等级)，软降 1 分，硬降每行 2 分
- **等级进阶** — 速度从 800ms 指数级降至 50ms 每 tick
- **状态机** — 菜单 → 游戏中 → 暂停 → 游戏结束

## 快速开始

```bash
cargo run
```

## 按键操作

| 按键 | 动作 |
|------|------|
| ← → | 左右移动 |
| ↑ | 顺时针旋转 |
| Z | 逆时针旋转 |
| ↓ | 软降 |
| 空格 | 硬降 |
| P | 暂停/继续 |
| Q | 退出 |
| Enter | 开始/确认 |

## 技术栈

- **Rust** — 系统级编程语言
- **ratatui** 0.29 — 终端 UI 渲染
- **crossterm** 0.28 — 终端输入与控制
- **rand** 0.8 — 随机数生成

## 项目结构

```
src/
├── main.rs       — 终端初始化 + 60fps 事件循环
├── app.rs        — 状态机（菜单/游戏中/暂停/游戏结束）
├── game.rs       — 核心逻辑：重力、锁定延迟、计分
├── board.rs      — 10×20 网格、碰撞检测、消行
├── piece.rs      — 7 种方块、SRS 旋转表、墙踢偏移
├── bag.rs        — 7-bag 随机生成器
├── ui.rs         — ratatui 渲染：棋盘、侧面板、菜单、覆盖层
├── input.rs      — crossterm 按键 → 动作映射
└── constants.rs  — 网格尺寸、时序、计分常量
```
